# BlogWeb

The full stack Blogging platform with user authentication.
Technology Involved : Django(Python Backend), HTML, CSS, JavaScript, Bootstrap & SQLite3 as database

Ckeditor is used for post content creation

2 modules :
1) Blog (for blogpost)
2) User (for users)
